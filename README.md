# QRShare — Instant Local Network File Sharer

A fast and modern desktop application designed to share files seamlessly across devices on your local Wi-Fi network using QR codes. Ever needed to send a file from your PC to your phone/another PC quickly without dealing with cables, slow cloud uploads, or messaging apps?
This project is a desktop application that turns your computer into a local file-sharing hub. One can easily share a file/an entire folder with other devices using this application. The other devices on the same network can download your files without uploading to any external cloud.

---

## Features

*  **Lightning-Fast Sharing**: Shares files directly over your Local Area Network (LAN) at maximum Wi-Fi speeds.
*  **Zero-Configuration Setup**: Generates a QR code instantly—scan with any phone camera to access shared files.
*  **Folder & File Support**: Easily select and browse individual files or entire directory trees.
